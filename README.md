## 🚀 Getting Started

### Prerequisites

* Node.js v18 or higher
* npm (included with Node.js)

### Installation

```bash
# Clone the repository
git clone https://github.com/khanirfan18/finBoard.git

# Navigate into the project
cd finBoard

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open:

```text
http://localhost:5173
```
